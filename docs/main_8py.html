Detailed Description

This file contains a modified version of JR Ridgely's basic_task.py (https://github.com/spluttflob/ME405-Support) that creates tasks for running two separate motors step responses at the same time.

This file contains all the functions printing out a graph of the encoder position to help us visualize the runtime nature of the motor.
